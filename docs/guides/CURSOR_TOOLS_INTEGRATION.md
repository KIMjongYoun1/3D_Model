```
┌─────────────────────────────────────────────────────────────────────────────┐

│     → Chat: 선택한 모델                                                     │
│     → Composer: 멀티파일 편집                                               │

└─────────────────────────────────────────────────────────────────────────────┘

## 👥 협업 시 환경 통일 가이드

### 1. 필수 설정 파일 공유

```
프로젝트 루트에 포함할 설정 파일:

📁 project-root/
├── .vscode/
│   ├── settings.json      # 에디터 설정
│   ├── extensions.json    # 권장 확장
│   └── launch.json        # 디버깅 설정
├── .editorconfig          # 코드 스타일
├── .prettierrc            # Prettier 설정
├── .eslintrc.js           # ESLint 설정
└── pyproject.toml         # Python 설정 (ruff, black 등)
```

### 2. .vscode/settings.json 예시

```json
{
  "editor.formatOnSave": true,
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "[python]": {
    "editor.defaultFormatter": "ms-python.black-formatter"
  },
  "python.linting.enabled": true,
  "typescript.preferences.importModuleSpecifier": "relative",
  "files.exclude": {
    "**/__pycache__": true,
    "**/.pytest_cache": true,
    "**/node_modules": true
  }
}
```

### 3. .vscode/extensions.json 예시

```json
{
  "recommendations": [
