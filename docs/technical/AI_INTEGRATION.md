# AI 모듈 연동 가이드

> **상태**: 준비 중 (아직 연동하지 않음)  
> **목적**: AI 모델 연동 시 검증 및 테스트 방법 문서화

## 개요

이 문서는 AI 모델 연동 시 필요한 검증 방법과 테스트 절차를 정리합니다.  
실제 연동이 시작되면 이 문서에 상세 내용을 추가합니다.

## 연동 예정 모델

### 1. IDM-VTON (Virtual Try-On)
- **용도**: 의상 가상 착용
- **입력**: 사람 이미지 + 옷 이미지
- **출력**: 착용 합성 이미지

### 2. Segment Anything (SAM)
- **용도**: 의상 세그멘테이션
- **입력**: 옷 이미지
- **출력**: 세그멘테이션 마스크

### 3. MediaPipe
- **용도**: 얼굴 랜드마크 및 메시 생성
- **입력**: 얼굴 이미지
- **출력**: 얼굴 메시 좌표

## 검증 방법 (연동 후 추가 예정)

### 로컬 테스트 환경
- GPU 환경 확인
- 모델 다운로드 및 로드 테스트
- 샘플 이미지로 추론 테스트

### API 엔드포인트 테스트
- FastAPI 엔드포인트 생성
- 요청/응답 형식 검증
- 에러 핸들링 테스트

### 성능 테스트
- 처리 시간 측정
- 메모리 사용량 확인
- 동시 요청 처리 테스트
