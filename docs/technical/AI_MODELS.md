## 🎯 Virtual Try-On 모델 비교

### 추천 모델: IDM-VTON ⭐

| 항목 | 내용 |
|------|------|
| **GitHub** | https://github.com/yisol/IDM-VTON |
| **라이선스** | Apache 2.0 (상용 가능) |
| **특징** | Diffusion 기반, 고품질 결과물 |
| **입력** | 사람 이미지 + 옷 이미지 |
| **출력** | 착용 합성 이미지 |
| **GPU 요구** | 24GB VRAM (RTX 4090) |
| **처리 시간** | 15~30초/장 |

### 대안 모델

| 모델 | 장점 | 단점 | 라이선스 |
|------|------|------|----------|
| **OOTDiffusion** | 고품질 | 연구용 | 비상업 |
| **HR-VITON** | 가벼움 | 품질 중간 | MIT |
| **StableVTON** | Stable Diffusion 기반 | 새로움 | Apache 2.0 |

## 🔧 AI 파이프라인 구성

```
┌─────────────────────────────────────────────────────────────────┐
│                     Try-On 파이프라인                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  [입력]              [처리]                    [출력]            │
│                                                                  │
│  옷 이미지 ─────▶ SAM ─────▶ 옷 세그먼트                        │
│                       │                                          │
│                       ▼                                          │
│  마네킹/아바타 ───▶ IDM-VTON ─────▶ 착용 이미지                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                    아바타 생성 파이프라인                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  얼굴 사진 ─────▶ MediaPipe ─────▶ Face Mesh                    │
│                       │                                          │
│                       ▼                                          │
│  체형 정보 ─────▶ 아바타 생성 ─────▶ 3D 모델                     │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
