# 📚 진짜 지식(Ground Truth) 운영 및 API 연동 전략

본 문서는 Quantum Studio의 AI 에이전트가 참고하는 전문 지식 데이터의 신뢰성을 보장하기 위한 운영 방침과 기술적 구현 방안을 기술합니다.

## 1. 지식 데이터 확보 원칙
- **신뢰성**: 국가기관 및 공인된 기술 표준 기구의 공식 데이터만을 채택합니다.
- **비용**: 본 프로젝트에서는 **100% 무료 오픈 API**만을 사용하여 운영 비용을 0원으로 유지합니다.
- **최신성**: 법령 개정 및 표준 업데이트에 대응하기 위해 자동/수동 업데이트 체계를 병행합니다.

## 2. 주요 지식 소스 및 API 정보

| 도메인 | 소스 기관 | API 명칭 | 비용 | 비고 |
|:---:|:---|:---|:---:|:---|
| **법률/세무** | 법제처 | 국가법령정보센터 오픈API | **무료** | 대한민국 모든 법령 및 판례 |
| **금융/회계** | 금융감독원 | DART 오픈 API | **무료** | 상장사 재무제표 및 공시 정보 |
| **금융/통계** | 한국은행 | 경제통계시스템(ECOS) API | **무료** | 산업별 평균 지표 및 금리 정보 |
| **IT 보안** | KISA | 보안 가이드라인 (Scraping) | **무료** | ISMS-P 및 클라우드 보안 표준 |

## 3. 지식 업데이트 메커니즘

### A. 자동 배치 업데이트 (Scheduled Update)
- **주기**: 매주 일요일 AM 02:00 (시스템 부하 최소화 시간대)
- **로직**: 
  1. `KnowledgeBase`에 등록된 핵심 법령/지표 리스트를 순회합니다.
  2. 각 API를 호출하여 `last_updated_date` 또는 `version`을 비교합니다.
  3. 변경 사항 발견 시 새로운 지식 버전을 생성하고 관리자에게 알림을 보냅니다.

### B. 수동 즉시 업데이트 (Manual Update)
- **대상**: 긴급한 법령 개정이나 관리자가 즉시 반영을 원하는 특정 도메인 지식.
- **방법**: 관리자 페이지(`/admin/knowledge`)에서 'API 즉시 수집' 버튼 클릭 또는 법령명 직접 검색.

## 4. API 키 발급 가이드 (무료)

1. **국가법령정보센터**: [open.law.go.kr](https://open.law.go.kr) 회원가입 후 '인증키 신청' (즉시 발급)
2. **금융감독원 DART**: [opendart.fss.or.kr](https://opendart.fss.or.kr) 회원가입 후 'API Key 신청' (이메일 인증 후 발급)
3. **한국은행 ECOS**: [ecos.bok.or.kr](https://ecos.bok.or.kr) 'Open API' 메뉴에서 신청

## 5. 데이터 검증 프로세스
1. **수집(Collected)**: API를 통해 원본 데이터 수집.
2. **정제(Refined)**: AI가 이해하기 쉬운 시각화 규칙(Visual Rules)으로 변환.
3. **검수(Verified)**: 관리자가 최종 내용을 확인하고 `is_active=true`로 승인.
4. **활용(Applied)**: AI 에이전트가 분석 시 해당 지식을 참고(RAG).
