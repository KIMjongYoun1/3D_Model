# 🗺️ Quantum Studio 개발 로드맵

Quantum Studio는 비정형 데이터를 3D 공간에 지능적으로 도식화하는 차세대 시각화 플랫폼으로의 발전을 목표로 합니다.

> **최종 업데이트**: 2026-02-24

---

## 🚀 Phase 1: Core MVP (현재 단계)
**목표**: 핵심 3D 엔진 구축 및 멀티 백엔드 인증 체계 완성
- [x] **멀티 백엔드 아키텍처**: Java(인증/결제) + Python(AI/매핑) 구조 확립
- [x] **3D 시각화 엔진**: Three.js 기반 노드-링크 렌더링 구현
- [x] **소셜 인증**: 네이버 로그인 연동 및 JWT 이중 토큰(Access/Refresh) 시스템
- [x] **디자인 시스템**: Apple-Clean White 테마 및 공통 UI 컴포넌트 구축
- [x] **데이터 매핑**: JSON 데이터를 3D 노드로 자동 변환하는 기본 엔진
- [x] **Admin 운영 기능**: 회원관리(정지/탈퇴), 거래관리(결제 취소), 구독관리(취소), 플랜관리, 약관관리, 매출 대시보드

---

## 📊 Phase 2: Advanced Visualization (진행 예정)
**목표**: 시각적 피드백 강화 및 사용자 인터랙션 고도화
- [ ] **지능형 카메라**: 데이터 구조에 따른 최적의 시점 자동 추적 (Auto Focus 고도화)
- [ ] **다양한 렌더링 모드**: 바(Bar), 다이어그램(ERD), 모놀리스(Monolith) 모드 전환 안정화
- [ ] **실시간 검색**: 대규모 노드 중 특정 데이터 즉시 탐색 및 포커스 기능
- [ ] **온보딩 가이드**: 인터랙티브한 단계별 조작 가이드 구현

---

## 🤖 Phase 3: Analysis Engine (진행 예정)
**목표**: AI 분석 기능 강화 및 데이터 처리 자동화
- [ ] **문서 분석**: PDF, Excel, TXT 파일 업로드 및 AI 텍스트 추출 연동
- [ ] **상관관계 엔진**: DB 기반 규칙을 활용한 자동 관계 생성 엔진 활성화
- [ ] **하이브리드 AI 라우팅**: 비용 최적화를 위한 클라우드(Gemini)와 로컬 모델 분기 처리
- [ ] **데이터 히스토리**: 과거 분석 프로젝트 저장 및 불러오기 기능 (사이드바)

---

## 🌐 Phase 4: Platform Expansion (장기 목표)
**목표**: 서비스 상용화 및 플랫폼 생태계 확장
- [ ] **결제 시스템 실전 연동**: 실제 PG사 연동을 통한 프로 플랜 구독 기능
- [ ] **협업 기능**: 시각화된 3D 맵을 공유하고 공동 편집하는 기능
- [ ] **B2B API**: 외부 서비스에서 Quantum 엔진을 사용할 수 있는 API 제공
- [ ] **모바일/WebXR**: 모바일 앱 및 VR 기기 대응 시각화 레이아웃

---

## 🛠 기술 스택 (Tech Stack)
- **Frontend**: Next.js 14, Three.js, Tailwind CSS
- **Java Backend**: Spring Boot 3.2, Java 21, JPA, Flyway
- **Python Backend**: FastAPI, Python 3.12, SQLAlchemy, Alembic
- **AI**: Google Gemini Pro, Local LLM (TBD)
- **Database**: PostgreSQL, Redis
